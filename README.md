# Visão Geral
Este repositório contém exemplos de arquivos XML, JSON e HTML para referência sobre o processo de transporte de dados sobre o protocolo HTTP.

Projeto construído para a disciplina **Desenvolvimento de WebServices** do curso **Tecnologia em Sistemas para Internet (TSI) do Senac-SP**.
